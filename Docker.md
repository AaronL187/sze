# Használt docker, docker compose utasítások

## Docker mappába belépés
```
cd .docker
```

## Docker konténer indítása
```
docker-compose up -d
```

## Docker konténer leállítása
```
docker-compose down
```

## Docker konténerek listázása
```
docker ps
```

## Docker konténerbe belépés
```
docker exec -it KONTÉNERNEVE bash
```
